# CalculatorApp

## Архитектура

Приложение построено по принципу разделения ответственности с четким разделением на слои:

### Пакеты:
- **ui** — Activity, Fragment, Adapterы. Отвечает за отображение и взаимодействие с пользователем
- **logic** — Чистая бизнес-логика (Evaluator). Не зависит от Android SDK
- **data** — Работа с базой данных (Room Database, DAO)
- **model** — Data classes (HistoryItem)

### Технологии:
- **Material Design 3** — современный UI
- **Room** — локальное хранение истории вычислений
- **Coroutines + Flow** — асинхронные операции
- **ViewBinding** — типобезопасный доступ к View
- **mXparser** — математический парсер для вычислений

### Особенности:
- Поддержка сложных выражений со скобками
- История вычислений с временными метками
- Navigation Drawer для просмотра истории
- Ripple-эффект при нажатии на кнопки
- Моноширинный шрифт для корректного отображения цифр

## Требования:
- Android Studio Hedgehog | 2023.1.1 или новее
- JDK 21
- Android SDK 34
- Kotlin 1.9.22

## Сборка:
1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Соберите проект: Build → Make Project (Ctrl+F9)
4. Для создания APK: Build → Build Bundle(s) / APK(s) → Build APK(s)
